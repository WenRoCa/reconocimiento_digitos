# 🔢 Reconocedor de Dígitos con Deep Learning

![Python](https://img.shields.io/badge/Python-3.10-blue)
![TensorFlow](https://img.shields.io/badge/TensorFlow-Deep%20Learning-orange)
![Status](https://img.shields.io/badge/Status-Completed-success)

Sistema de reconocimiento de dígitos escritos a mano usando redes neuronales.

---

## 📌 Descripción

Este proyecto utiliza el dataset MNIST para entrenar modelos capaces de reconocer números del 0 al 9.

---

## 🧠 Tecnologías

- Python
- TensorFlow / Keras
- NumPy
- Matplotlib

---

## 🤖 Modelos

- Red Pequeña
- Red Mediana
- Red Grande

---

## 📊 Resultados

| Modelo | Precisión |
|--------|----------|
| Pequeña | 92% |
| Mediana | 96% |
| Grande | 97% |

🏆 Mejor modelo: Red Grande

---

## ▶️ Ejecución

```bash

pip uninstall numpy tensorflow keras -y
pip install tensorflow==2.15.0 keras==2.15.0 numpy==1.26.4

python reconocimiento_digitos.py

```
